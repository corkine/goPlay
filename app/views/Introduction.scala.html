短链接跳转服务

此服务用来将 go.mazhangjing.com/?? 的网址重定向到其他站点，可类似看做短连接服务或者网页收藏服务。

服务保证：所有 go.mazhangjing.com/?? 指向的链接将在添加到系统内的 5 年内有效，但不保证指向的 URL 有效。

了解更多

关键词跳转：对于关键词 home 的链接, 直接访问 go.mazhangjing.com/home 即可跳转到指定链接。
序列号跳转：对于序列号 233 的链接，直接访问 go.mazhangjing.com/id/233 即可跳转到指定链接。

基本：通过 GET /add 方法

通过 GET go.mazhangjing.com/add，传入参数 keyword、redirectURL、note（可选）以添加记录，返回 JSON 格式文件。

比如 GET go.mazhangjing.com/add?keyword=home&redirectURL=www.mazhangjing.com 可创建 go.mazhangjing.com/home 指向 www.mazhangjing.com 的链接。注意，此处 redirectURL 可以加 https:// 或者 http://，也可以不加。

更多：参见网站路由

注意，对于受保护 API 而言，需要通过用户名和密码的验证，验证方式包括通过浏览器 Basic 认证、通过 token Query 参数、通过 user 和 password Query 参数，token 的获取可访问 /token 获取。

本程序使用 Play Scala with sbt 开发，数据存储使用 H2 嵌入式数据库，数据库访问采用 Slick FRM 框架，前端界面采用 Bootstrap 和 Vue with Ant Design(管理部分) 构建。
